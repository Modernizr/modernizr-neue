Modernizr 3.11.0

Posted by Rejas, on April 27, 2020

We might be living in interesting times, two releases in one month :-) Getting a small release out the door before work on v4 starts. Stay tuned for that...

New Feature Detect(s)

@dabretin continued his addition of es6 and beyond:

  • ES6 - class #2532
  • ES8 - object #2533
  • TextEncoder and TextDecoder #2531

@MarkelFe added:

  • av1 and h265 codec detection #2534

Fixes

  • @Sembiance fixed an issue with AVG Secure browser throwing exceptions when using toDataURL when Anti-Fingerprinting is turned on #2538
  • @MarkelFe added some missing caniuse mappings #2535

Miscellaneous

  • Update computedStyle usage #2528
  • Cleanup property tags #2542
  • Change property name for es6 class test 1baf805
